## کلاؤڈ-نیٹو لیب: فزیکل ہارڈ ویئر کا ایک متبادل

اگر آپ کو RTX-ان ایبلڈ ورک سٹیشنز تک رسائی حاصل نہیں ہے، تو کلاؤڈ-نیٹو لیب ایک قابل عمل متبادل ہے۔ یہ طریقہ کلاؤڈ بیسڈ انسٹینسز (جیسے AWS RoboMaker یا NVIDIA کی Omniverse کے لیے کلاؤڈ ڈیلیوری) پر انحصار کرتا ہے اور تیزی سے تعیناتی کے لیے یا اگر آپ کے پاس کم طاقتور لیپ ٹاپ ہے تو مثالی ہے۔

:::info
ایک "فزیکل AI" لیب بنانا ایک بڑی سرمایہ کاری ہے۔ آپ کو ایک فزیکل **آن-پریمیس لیب** (زیادہ سرمائے کے اخراجات) بنانے یا ایک **کلاؤڈ-نیٹو لیب** (زیادہ آپریٹنگ اخراجات) چلانے کے درمیان انتخاب کرنا پڑے گا۔
:::

### "ایتھر" لیب (کلاؤڈ-نیٹو)

یہ آپشن بہترین ہے:
-   تیزی سے تعیناتی کے لیے
-   کمزور لیپ ٹاپ والے طلباء کے لیے

#### 1. کلاؤڈ ورک سٹیشنز (AWS/Azure)

طاقتور پی سی خریدنے کے بجائے، آپ کلاؤڈ انسٹینسز کرائے پر لے سکتے ہیں۔

-   **انسٹینس ٹائپ:** AWS g5.2xlarge (A10G GPU، 24GB VRAM) یا g6e.xlarge۔
-   **سافٹ ویئر:** Omniverse Cloud پر NVIDIA Isaac Sim (ایک مخصوص Amazon Machine Image - AMI کی ضرورت ہے)۔

#### لاگت کا حساب کتاب

-   **انسٹینس لاگت:** تقریباً $1.50 فی گھنٹہ (اسپاٹ اور آن ڈیمانڈ انسٹینسز کے مرکب کا استعمال کرتے ہوئے)۔
-   **استعمال:** 10 گھنٹے/ہفتہ × 12 ہفتے = 120 گھنٹے۔
-   **اسٹوریج:** تقریباً $25 فی سہ ماہی (اپنے ماحول کو محفوظ کرنے کے لیے EBS والیومز کے لیے)۔
-   **کُل کلاؤڈ بل:** تقریباً $205 فی سہ ماہی۔
