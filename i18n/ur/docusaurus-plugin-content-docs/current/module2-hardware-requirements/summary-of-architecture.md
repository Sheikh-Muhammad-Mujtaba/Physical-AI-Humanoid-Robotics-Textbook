## لیب کے فنِ تعمیر کا خلاصہ

اس کورس کو کامیابی سے پڑھانے کے لیے، آپ کے لیب کے بنیادی ڈھانچے میں مندرجہ ذیل اجزاء شامل ہونے چاہیئں:

| جزو | ہارڈویئر | کردار |
| :--- | :--- | :--- |
| **سِم رِگ** | RTX 4080 اور Ubuntu 22.04 کے ساتھ PC | Isaac Sim، Gazebo، Unity چلاتا ہے، اور LLM/VLA ماڈلز کو تربیت دیتا ہے۔ |
| **ایڈج برین** | Jetson Orin Nano | "Inference" اسٹیک چلاتا ہے۔ طلباء اپنا کوڈ یہاں deploy کرتے ہیں۔ |
| **سینسرز** | RealSense کیمرہ + Lidar | Jetson سے منسلک ہوتے ہیں تاکہ حقیقی دنیا کا ڈیٹا AI کو فراہم کریں۔ |
| **ایکچوئیٹر** | Unitree Go2 یا G1 (مشترکہ) | Jetson سے موٹر کمانڈز وصول کرتا ہے۔ |
