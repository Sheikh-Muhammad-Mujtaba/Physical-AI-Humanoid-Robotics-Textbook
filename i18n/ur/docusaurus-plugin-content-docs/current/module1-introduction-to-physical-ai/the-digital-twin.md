## ڈیجیٹل ٹوئن: Gazebo اور Unity کے ساتھ سمولیشن

حقیقی دنیا میں ایک روبوٹ کو تعینات کرنے سے پہلے، اس کے طرز عمل کو ایک سمولیٹڈ ماحول میں جانچنا اور اس کی توثیق کرنا بہت اہم ہے۔ یہی **ڈیجیٹل ٹوئن** کا تصور ہے — کسی طبعی چیز یا سسٹم کی ایک ورچوئل نمائندگی۔

### فزکس سمولیشن اور ماحول کی تشکیل

اس ماڈیول میں، آپ دو طاقتور سمولیشن ٹولز کا استعمال کرنا سیکھیں گے:

-   **Gazebo:** ایک مضبوط فزکس سمیولیٹر جو آپ کو پیچیدہ ماحول اور روبوٹ کے باہمی تعاملات کا ماڈل بنانے کی اجازت دیتا ہے۔ آپ فزکس، کشش ثقل، اور تصادم کو سمولیٹ کرنا سیکھیں گے۔
-   **Unity:** ایک اعلیٰ فیڈیلیٹی رینڈرنگ انجن جو فوٹو ریئلسٹک بصری فراہم کرتا ہے اور انسان-روبوٹ کے باہمی تعامل کے بھرپور منظرناموں کو ممکن بناتا ہے۔

### حواس کی سمولیشن

ایک روبوٹ اتنا ہی اچھا ہے جتنا کہ وہ دنیا کو سمجھنے کی صلاحیت رکھتا ہے۔ آپ مختلف قسم کے سینسرز کو سمولیٹ کرنا سیکھیں گے، جن میں شامل ہیں:

-   **LiDAR:** ماحول کے 3D نقشے بنانے کے لیے۔
-   **Depth Cameras:** گہرائی اور فاصلے کو سمجھنے کے لیے۔
-   **IMUs (Inertial Measurement Units):** سمت اور سرعت کو ٹریک کرنے کے لیے۔

### ہفتہ وار تقسیم (ہفتے 6-7)

-   **Gazebo Simulation Environment:** ایک Gazebo سمولیشن ماحول کو ترتیب دینا اور کنفیگر کرنا سیکھیں۔
-   **Robot Descriptions:** اپنے روبوٹ کی وضاحت کے لیے URDF (Unified Robot Description Format) اور SDF (Simulation Description Format) کو سمجھیں۔
-   **Physics and Sensor Simulation:** Gazebo میں فزکس اور سینسرز کو سمولیٹ کرنے کی تفصیلات میں گہرائی سے جائیں۔
-   **Unity for Visualization:** اعلیٰ معیار کی روبوٹ ویژولائزیشن کے لیے Unity کے استعمال کا تعارف حاصل کریں۔
